# مبات - Mabat 🌊

منصة حجز المنتجعات الساحلية الفاخرة في المملكة العربية السعودية

## تشغيل المشروع محلياً

```bash
npm install
npm run dev
```

افتح المتصفح على: http://localhost:5173

## بناء للإنتاج

```bash
npm run build
npm run preview
```

## ميزات المنصة

- 13 منتجعاً ساحلياً حقيقياً بأسعار من التقرير الرسمي 2026
- نظام تسجيل دخول وإنشاء حساب (localStorage)
- نظام حجز كامل مع حساب الأسعار والليالي
- نظام تقييمات مخصص لكل منتجع
- تصفية وبحث متقدم (المنطقة، السعر، الترتيب)
- لوحة حساب المستخدم مع سجل الحجوزات
- تصميم RTL عربي كامل
- Liquid Glass design system
- Framer Motion animations
- Fully responsive (375px → 1440px)
